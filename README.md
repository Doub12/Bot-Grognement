# Bot-Grognement

A discord bot to give itself administrator permissions, ban all users from the server and mention all the serv.

## Installation

Npm to install module.

```bash
npm install 
```

## Commands
Give itself administrator permissions
```bash
!perm
```
Ban all users
```bash
!ban
```
Mention all the serv
```bash
!mention
```
Leave the server
```bash
!leaveserver
```
